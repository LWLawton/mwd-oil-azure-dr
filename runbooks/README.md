## Runbook Review Policy

All runbooks are reviewed:
- **Quarterly** — scheduled review by SOC Lead
- **Post-incident** — after any P1 or P2 incident that used the runbook
- **After significant environment changes** — major architecture or tooling changes

Runbook updates require CISO approval before merging to `main`.
